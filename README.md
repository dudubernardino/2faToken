# 2FA Token CLI

Explore the 2FAToken CLI Tool, tailored for developers to efficiently test 2FA tokens integration.

## 1. Description

The 2FAToken SDK offers a javascript library, simplifying interactions with the 2FAToken API for seamless integration.

## 2. Installation

To install the Iniciador SDK, run the following command:

```bash
npm install -g 2fatoken
```

## 3. Usage

To use the cli use the command below:

```bash
2fatoken -s SECRET
```
